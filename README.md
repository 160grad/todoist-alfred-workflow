# About this Alfred workflow

This Alfred workflow (for Alfred v.2+) adds a new tasks to your [Todoist](www.todoist.com) inbox. Just type 'todo' (or change the keyword) followed by the task and hit 'enter'. Voil√†, you just added a todo to your Todoist inbox.


# Installation

1. Download the .zip archive
2. Double click to install the Alfred workflow
3. In the workflow overview, edit the shell script and add you API key 
4. Add a shortcut for fast access
